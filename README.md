# 简介

土豆计时器 ( todo timer ) 是一款基于 uni-app 的开源跨平台应用。
土豆计时器 主要包含两个模块，单次计时器和循环计时器。单次计时器完成后直接退出。循环计时器可编辑计时器组和循环次数，完成所有步骤后退出。
土豆计时器 可用于厨房计时、HIIT健身计时、行为习惯培养计时等场景。
